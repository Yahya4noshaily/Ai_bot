طريقة تشغيل البوت على سيرفرك:

1. افتح السيرفر وسوِ:
   git clone https://github.com/Yahya4noshaily/Ai_bot.git

أو حمّل المجلد المضغوط هذا وافكّه:

2. ادخل المجلد:
   cd eo_auto_trader

3. شغّل البوت:
   python3 main.py

4. لإيقافه مؤقتاً:
   افتح config.py وغير:
   AUTO_TRADE = False

واعمل إعادة تشغيل للملف main.py
